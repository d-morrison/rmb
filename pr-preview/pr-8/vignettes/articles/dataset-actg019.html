2 RMB2e analysis workflow

2.1 Research question

Can zidovudine treatment reduce the hazard of AIDS/death after accounting for baseline CD4 cell count?

RMB2e references: trial context and survival-data motivation (p. 224), Cox modeling details and bootstrap CI discussion (p. 267), coding sensitivity exercises (p. 278).

2.2 Causal assumptions (conceptual DAG)

Code
flowchart LR
  RX[ZDV treatment (rx)] --> T[Time to AIDS/death]
  CD4[Baseline CD4] --> T
  CD4 --> RX

flowchart LR
  RX[ZDV treatment (rx)] --> T[Time to AIDS/death]
  CD4[Baseline CD4] --> T
  CD4 --> RX

Because ACTG 019 was randomized, treatment is unconfounded in expectation; CD4 is included as a strong prognostic baseline covariate (RMB2e p. 268).

2.4 Step 1: Model specification

RMB2e model: Cox proportional hazards model with treatment (rx) and baseline CD4 (cd4) as predictors (p. 267).

Code
cox_formula <- survival::Surv(days, cens) ~ rx + cd4
cox_formula
#> survival::Surv(days, cens) ~ rx + cd4

2.7 Step 4: Inference

RMB2e discusses Wald and bootstrap interval interpretation for this model family (p. 267).

Code
hr <- exp(stats::coef(fit_cox))
ci <- exp(stats::confint(fit_cox))
out <- data.frame(
  term = names(hr),
  hazard_ratio = unname(hr),
  conf_low = ci[, 1],
  conf_high = ci[, 2],
  p_value = summary(fit_cox)$coefficients[, "Pr(>|z|)"]
)
out
#>     term hazard_ratio  conf_low conf_high      p_value
#> rx    rx    0.4560671 0.2567927 0.8099809 7.381988e-03
#> cd4  cd4    0.9934464 0.9910395 0.9958591 1.079987e-07

2.8 Scientific conclusions

In this RMB2e ACTG 019 reproduction, treatment assignment to ZDV is associated with a lower hazard of AIDS/death, while higher baseline CD4 is associated with lower hazard. These are the same scientific directions emphasized in Chapter 6 (pp. 224, 267).
